# 👑 Crorepati Quiz Game

A fully interactive **KBC / Crorepati inspired quiz application** with modern UI, prize money ladder, timer system, lifelines, leaderboard tracking, sound effects, and responsive gameplay experience.

The project recreates the excitement of quiz competitions while keeping the implementation beginner-friendly using only **vanilla HTML, CSS, and JavaScript**.

---

## 🚀 Features

### 🎮 Quiz Gameplay System
- Interactive multiple-choice quiz gameplay
- Dynamic question loading system
- Auto next-question progression
- Win / Lose game flow
- Real-time gameplay updates
- Game over & victory screens
- Restart functionality

### 💰 Money Ladder System
- KBC-style prize money structure
- Dynamic current level highlighting
- Prize progression system
- Safe checkpoint levels
- Visual money tree interface

Safe Levels:
- ✅ Rs 10,000
- ✅ Rs 320,000

---

## 👤 Player Profile System

- Username input system
- Avatar selection before game start
- Live player profile display
- Dynamic player identity during gameplay

Available avatars:

- 👑 King
- 🎯 Pro
- 🚀 Master
- 🔥 Legend

---

## 🛟 Lifeline System

### 50:50 Lifeline
- Removes two incorrect answers
- One-time use only
- Smart elimination logic

### 🔄 Flip Question
- Skip current question
- Load next available question
- Single-use functionality

### 👥 Audience Poll
- Audience suggestion system
- Displays recommended answer
- Interactive hint mechanism

---

## ⏱ Timer Gameplay System

- 30-second countdown timer
- Auto timer reset for each question
- Real-time timer updates
- Automatic loss when timer ends
- Dynamic timer UI display

---

## 🏆 Leaderboard System

- LocalStorage score persistence
- Stores player data automatically
- Saves:

  - Username
  - Avatar
  - Score

- Automatic score sorting
- Top player display
- Ranking system
